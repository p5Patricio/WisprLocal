# WisprLocal

Proyecto de transcripción local utilizando Whisper y Python, con soporte para GPU y atajos de teclado para el micrófono.

## Archivos Principales
- `mvp_local.py`: Script principal del MVP de transcripción.
- `test_transcripcion.py`: Pruebas de transcripción.
- `prueba_mic_hotkey.py`: Pruebas de grabación de micrófono mediante atajos de teclado.
- `verificar_gpu.py`: Utilidad para verificar la disponibilidad de la GPU para aceleración.
- `lanzador.vbs`: Script lanzador para Windows.
